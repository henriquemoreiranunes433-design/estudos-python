# 🐍 Estudos de Python 

Este repositório contém meu progresso e projetos desenvolvidos durante o curso completo de Python. O foco é aprender desde a lógica fundamental até a automação de processos e criação de interfaces gráficas.

## 🚀 Projetos em Destaque

### 🧮 Calculadora com PySide6
Localizada em: `/calculadora_pyside`
- Interface gráfica moderna criada com Python e PySide.
- Foco em manipulação de eventos e lógica de interface.

### 🏦 Sistema Bancário
Localizada em: `/sistema_bancario`
- Aplicação prática de Orientação a Objetos (POO).
- Simulação de operações de depósito, saque e extrato.
- Integração com conceitos de Banco de Dados.

## 📁 Organização do Repositório
